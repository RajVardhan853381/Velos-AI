Velos

Decentralized Blind Hiring Protocol

What's the deal?

Build AI agents that actually talk to each other and solve real problems together. Think beyond solo AI—we're talking networks, collaboration, decentralized intelligence.

Velos solves two critical hiring problems:

  1. Bias in AI Hiring → Anonymize data before evaluation
  2. Resume Fraud → Verify authenticity via technical questions
